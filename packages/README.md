# Packages

本目录包含前端监控系统的所有子模块：

## 子模块列表

- [monitor-core](./monitor-core) - 监控核心模块，提供基础功能和数据结构
- [monitor-browser](./monitor-browser) - 浏览器端监控 SDK
- [monitor-uniapp](./monitor-uniapp) - UniApp 应用监控 SDK
- [monitor-wxapp](./monitor-wxapp) - 微信小程序监控 SDK
- [monitor-server](./monitor-server) - 服务端数据接收和处理

每个子模块都有独立的 README.md 文件，详细介绍了其功能和使用方法。
